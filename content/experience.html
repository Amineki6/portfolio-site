Research Assistant @ Constructor University (01/2024 - 08/2024)
Developed a scalable movie recommendation system based on a microservices architecture, leveraging Docker, Kafka, Redis, PostgreSQL, and Spring Boot for efficient communication and data processing. Integrated Prometheus and Grafana for real-time system monitoring and performance evaluation. Optimized data flow using Kafka-based event-driven communication, ensuring seamless synchronization between services. Conducted system performance evaluation using Locust for load testing, achieving high request throughput and low-latency recommendation delivery.
The system includes a robust API gateway for secure and efficient request routing between services. A user activity tracking module was implemented to gather behavioral data, enhancing recommendation accuracy. Additionally, a rolling model update mechanism ensures seamless deployment of improved recommendation models without service interruptions, allowing continuous adaptation to evolving user preferences.
