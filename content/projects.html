AI-Based Bank Statement Aggregation System
Developed an advanced AI-powered system for automating the extraction and digitization of bank statements. This project leveraged large language models (LLMs) and OCR frameworks to process structured and unstructured financial data from variously formatted documents. By integrating GPT-4o with Vision capabilities, the system achieved over 90% accuracy in extracting transaction details, reducing reliance on traditional OCR methods and eliminating complex preprocessing steps.
Implemented a Docker-based architecture for seamless deployment and scalability, enabling parallel processing of financial documents. The extracted data was formatted into JSON and CSV and stored in a SQLite3 database for further analysis. The system demonstrated efficiency improvements over manual data entry while addressing challenges related to layout variability, multilingual support, and data correctness.

Developed a stock price prediction and trading agent by integrating time series forecasting with an LSTM model and reinforcement learning (RL) to optimize trading decisions. The system predicts future stock prices using an LSTM-based model and employs a DQN-based RL agent (powered by Stable-Baselines3) to execute trades in a custom Gym-compatible trading environment.
The project involves data preprocessing, model training, and evaluation, utilizing a historical stock market dataset from Kaggle. The LSTM model is trained to forecast stock prices, and its predictions serve as inputs for the RL agent, which learns to buy, sell, or hold stocks to maximize profitability. PillChat Tool - AI Agent
Developed PillChat, an AI-powered web application that provides users with reliable and accurate medication information through an interactive chatbot. Utilizing Retrieval-Augmented Generation (RAG), PillChat enhances response accuracy by leveraging patient information leaflets to deliver contextual and trustworthy answers. The platform supports multilingual queries, ensuring accessibility for a global audience. With its intuitive user interface, PillChat enables seamless medication lookup and intelligent conversational interactions, improving healthcare awareness and medication safety.
To power efficient and high-accuracy information retrieval, I integrated Pinecone, a scalable vector database, into PillChat. This enables fast and precise semantic search by indexing and retrieving relevant medication details from patient information leaflets. By leveraging Pinecone alongside Retrieval-Augmented Generation (RAG), PillChat delivers contextual and reliable answers, ensuring users receive accurate medication insights in real-time.
Fine-Tuned Model Summarizer
Developed a fine-tuned text summarization model deployed using Streamlit, enabling users to input text (up to 512 tokens) and receive concise summaries between 50-150 words. The project demonstrates the full lifecycle of fine-tuning a pre-trained transformer model, hosting it on Hugging Face, and deploying it via a web-based application.
The model was fine-tuned on Google Colab, leveraging Hugging Face Transformers for preprocessing and training. It includes a structured pipeline for dataset preprocessing, tokenization, and model optimization. The Streamlit app provides an intuitive interface where users can generate summaries in real time. The model and tokenizer are automatically fetched from Hugging Face upon startup, ensuring seamless deployment. Additionally, a virtual environment setup ensures smooth dependency management, making the project easy to reproduce and deploy.
Maximize Tips Revenue - Predictive Modeling
Developed Tip Predictor, a machine learning-powered tool designed to help workers maximize their tip earnings by predicting the best days to work. By analyzing historical data on tips, weather conditions, and order prices, the model provides actionable insights into how external factors influence tipping behavior. The goal is to enable workers to strategically schedule their shifts for higher earnings.
To achieve this, I implemented a RandomForestRegressor model trained on past tipping data, leveraging feature engineering to categorize weather conditions into intuitive groups such as 'Sunny', 'Cloudy', and 'Rainy'. Additionally, I integrated an external weather API for real-time data processing, ensuring dynamic and accurate predictions. The tool also features visualization capabilities, generating bar plots to illustrate tip trends across different days and weather conditions, making insights more accessible and actionable.
